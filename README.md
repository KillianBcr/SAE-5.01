## Installation / Configuration

### Install Symfony

```bash
composer install
```

### Install React App

```bash
npm install
```

### Run database server

```bash
docker-compose up -d
```

### Install database

```bash
composer db
```

### Build front assets with Webpack Encore

```bash
npm run build
```

### Run Symfony server

```bash
symfony server:start
```
